Mac OS X 安装 Tomcat 8

Dec 30, 2015

OS X 10.11 El Capitan 上安装 Tomcat 8

原链

自从OS X 10.7 不再预安装Java，我们先解决这个问题。

先决条件：Java

我在写这篇文章的时候，Java最新版本是8u66.请在这里 下载。

在Mac上使用dmg格式的安装包很容易安装JDK；安装后打开终端，输入

java -version

然后显示以下内容：

java version “1.8.0_66”

Java(TM) SE Runtime Environment (build 1.8.0_66-b17)

Java HotSpot(TM) 64-Bit Server VM (build 25.66-b17, mixed mode)

无论怎样，当你打开终端输入java -version，你要确保安装了Java7或更高版本。

安装Tomcat

以下是安装Tomcat的步骤

  1. 从 这里 下载apache-tomcat-8.0.30.tar.gz

  2. 打开并解压文件，可使用tar -zxvf命令

  3. 打开终端并移动解压后的文件到 /usr/local

    sudo mkdir -p /usr/local

    sudo mv ~/Downloads/apache-tomcat-8.0.30 /usr/local

  4. 增加符号连接

    sudo mkdir -p /usr/local

    sudo mv ~/Downloads/apache-tomcat-8.0.30 /usr/local

  5. 改变所有者

    sudo chown -R /Library/Tomcat

  6. 添加可执行权限

    sudo chmod +x /Library/Tomcat/bin/*.sh

运行Tomcat

~$ /Library/Tomcat/bin/startup.sh

~$ /Library/Tomcat/bin/shutdown.sh

打开浏览器，输入：http://localhost:8080

安装成功
